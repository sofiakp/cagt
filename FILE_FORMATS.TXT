Specification of file formats:

CAGT data format:
-----------------------------------------------
chr	start	stop	val1,val2,...,valN

Profiles list file:
-----------------------------------------------
profiles_file_absolute_path	peak_tag	signal_tag	cell_line	peak_original_filename	signal_original_filename	ylim_lower,ylim_upper	allow_profile_flipping?	number_of_bases_per_value	low_signal_cutoff_value

Signals list file:
-----------------------------------------------
signal_tag	cell_line	ylim_lower,ylim_upper	low_signal_cutoff_value	number_of_bases_per_value	num_bases_in_profile	filename(just the name, not a full path)

Peaks list file:
-----------------------------------------------
peak_tag	cell_line	flip?(True/False)	filename(just the name, not a full path)

profiles_file_absolute_path - This tells CAGT where to find the profiles file
peak_tag - A short name for the set of peaks used for this profiles file (i.e. Pol2). 
	In pairwise analysis (coming soon), CAGT only compares profiles with the same peak_tag. 
signal_tag - A short name for the signal used for this profiles file (i.e. H3K4me1)
cell_line - The cell line of this profiles file.  In pairwise analysis (coming soon), 
	CAGT only compares profiles files with the same peak_tag.
peak_original_filename - The filename of the peak file. This is used to generate 
	filenames, so it is important that it is unique.
peak_original_filename - The filename of the signal file. This is used to generate 
	filenames, so it is important that it is unique.
ylim_lower,ylim_upper - These define the limits on the Y axis of the plots generated.
allow_profile_flipping? - A value True or False, this tells CAGT whether or not 
	to flip profiles when clustering.
number_of_bases_per_value - The number of bases each value in the profile represents. 
	This is used in generating plots to determine the labels on the X axis.  
low_signal_cutoff_value - CAGT removes profiles with 90% quantile less than 
	low_signal_cutoff_value from the set of profiles to be clustered. 
	(This is done because correlation cannot be computed on profiles with extremely 
	low signal). For signal in units of sequencing fold change, 1 is a reasonable 
	value for this parameter.
